# Star Map App

Generate a star map for any place on Earth and any date/time.

## Features
- Location, date, time, and timezone controls
- Real astronomy-based star placement
- Constellation lines and optional labels
- Horizon / north marker / grid overlays
- Theme presets
- Shareable URL state
- SVG download

## Stack
- React
- Vite
- JavaScript
- SVG rendering

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
This first version is offline-first and uses a built-in star catalog plus seeded background stars for a richer sky view.
